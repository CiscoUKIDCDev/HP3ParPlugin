Cisco UCS Director SDK Bundle ReadMe
====================================

Prerequisites for Cisco UCS Director 5.4 and Above:
---------------------------------------------------

Install JDK 1.8 for the Cisco UCS Director Open Automation SDK bundle, Release 5.4.
	 

Instructions 
---------------

After you have downloaded and unzipped the SDK bundle, do the following:

1.  Import the project into Eclipse.
2.  Right-click on the project and choose Properties
3.  In the Properties wizard, click Java Compiler. 
4.  On the right side, choose Enable Project Specific Settings.
5.  Change Compiler Compliance Level to 1.8.
